Quids Pro Quo

A subscription website that provides access to barista training material for coffee shops. The site was built with Python, Django, Heroku, and PostgreSQL.


OKR Kit

A web app that allows users to create, update, and track Objectives and Key Results (OKRs). The app is built with NodeJS, MongoDB, and JWT. It was also an experiment in using Vanilla JS and CSS, so I did not use web server like HapiJS, a frontend framework like ReactJS, a CSS preprocessor like Less, or a module builder like Webpack. It was one of the most valuable experiences learning code that I've had.
